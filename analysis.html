Model Summaries:

  • Multinomial Logistic Regression

    • Multinomial Logistic Regression is used to find linear relationship between multiclass categories. Fundamental functions include the logit function (calculate probabilities) and assumption of variable independence during binary comparisons.

  • SVM

    • SVM is used to find linear separation between multiclass categories. Fundamental functions include maximum Euclidean distance, soft-margin, and a binary (one-versus-all) approach.

  • Multinomial Naive Bayes

    • Multinomial Naive Bayes classifier is used to find linear relationships of the features based on probabilities. Fundamental functions include maximum likelihood calculations and assumption of a single multinomial distribution of the variable.
